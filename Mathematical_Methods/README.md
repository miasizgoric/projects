**Course Objective:**
The aim of this course is the elaboration of concepts from linear algebra, number theory, polynomials, and real functions of multiple variables, which form the theoretical foundation for advanced topics in computer science and informatics.

**About the Project: Symmetric, Antisymmetric, and Hermitian Matrices**
This project explores the properties and applications of symmetric, antisymmetric, Hermitian, and anti-Hermitian matrices. The practical part of the project includes Python implementations for checking matrix properties, finding bases and dimensions of vector spaces over real ($\mathbb{R}$) and complex ($\mathbb{C}$) fields, and performing matrix decompositions. Additionally, the project demonstrates geometric interpretations of rotation matrices through 3D animations of geometric solids (rhombus, octahedron, spheres) generated using Blender.
