**Course Objective:**
The main objective of the course is to introduce students to the basics of domain knowledge modeling, knowledge representation, and automated reasoning as fundamental areas of Artificial Intelligence. Students gain knowledge in automata theory, propositional, and predicate logic, and apply this knowledge practically to model and solve problems in various business domains.

**About the Project: Modeling the Sokoban Puzzle**
This project explores the classic logic puzzle "Sokoban" through the lens of formal knowledge modeling. It defines the state space, operators, and critical constraints such as "deadlock" scenarios. The project compares two distinct programming paradigms for solving the puzzle: a declarative approach using Prolog (utilizing Depth-First Search for automated solving) and an imperative approach using Python (providing an interactive graphical implementation via Pygame).
