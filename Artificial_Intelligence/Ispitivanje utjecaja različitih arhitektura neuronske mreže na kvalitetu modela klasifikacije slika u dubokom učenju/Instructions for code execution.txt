MOBILENET
The code was developed by colleague Gabriela; Python version 3.12.9.
To run the MobileNetV2 architecture with the parameters from the lab exercises, you simply need to run the main.py program.
For the modified code with image enlargement, you need to uncomment lines 14 and 15:
#x_treniranje = tf.image.resize(x_treniranje,[96,96])
#x_testiranje = tf.image.resize(x_testiranje,[96,96])
and in line 32 input_shape=(32,32,3), change the input_shape to (96,96,3).
